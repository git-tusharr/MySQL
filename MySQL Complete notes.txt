MySQL Complete Notes

1. CREATE DATABASE
To create a new database in MySQL:
CREATE DATABASE databaseName;
Example:
CREATE DATABASE sql4pm;

2. DROP DATABASE
To delete an existing database:
DROP DATABASE databaseName;

3. SHOW DATABASES
To display all databases:
SHOW DATABASES;

4. CLEAR SCREEN
To clear the MySQL terminal:
SYSTEM CLS;

5. USE DATABASE
To select a specific database:
USE databaseName;

6. CHECK CURRENT DATABASE
SELECT DATABASE();

7. CREATE TABLE
CREATE TABLE tableName (
  columnName dataType(size),
  columnName dataType(size),
  ...
);
Example:
CREATE TABLE student (
  rollno INT(10),
  name VARCHAR(10),
  class VARCHAR(10),
  fees INT
);

8. SHOW TABLES
SHOW TABLES;

9. DESCRIBE TABLE
DESCRIBE tableName;
or
DESC tableName;

10. INSERT RECORD
INSERT INTO tableName (column1, column2, ...)
VALUES (value1, value2, ...);

11. INSERT MULTIPLE RECORDS
INSERT INTO tableName (column1, column2, ...)
VALUES (value1, value2, ...), (value1, value2, ...);

12. SELECT ALL RECORDS
SELECT * FROM tableName;

13. SELECT SPECIFIC COLUMNS
SELECT column1, column2 FROM tableName;

14. WHERE CLAUSE
SELECT * FROM student WHERE rollno = 101;

15. UPDATE STATEMENT
UPDATE tableName SET columnName = value WHERE condition;
Example: UPDATE student SET name = 'Tushar' WHERE rollno = 104;

16. DELETE STATEMENT
DELETE FROM tableName WHERE condition;

17. LOGICAL OPERATORS
AND - True if both conditions are true.
OR - True if any one condition is true.

18. BETWEEN OPERATOR
SELECT * FROM student WHERE rollno BETWEEN 113 AND 117;

19. IN / NOT IN OPERATORS
IN - Matches any value in the list.
NOT IN - Excludes all listed values.

20. DISTINCT KEYWORD
SELECT DISTINCT class FROM student;

21. LIKE OPERATOR
_ (underscore) - represents a single character.
% (percentage) - represents multiple characters.
Example:
SELECT * FROM student WHERE name LIKE 'S%';


22. ORDER BY
SELECT * FROM student ORDER BY name ASC;
SELECT * FROM student ORDER BY rollno DESC;


23. SQL DATE & TIME TYPES
DATE - YYYY-MM-DD
DATETIME - YYYY-MM-DD HH:MI:SS
TIMESTAMP - YYYY-MM-DD HH:MI:SS (auto-updated)
YEAR - YYYY or YY


24. STRING FUNCTIONS
CONCAT() - SELECT CONCAT('Mr. ', name) FROM student;
LOWER() - SELECT LOWER(name) FROM student;
UPPER() - SELECT UPPER(name) FROM student;
SUBSTR() - SELECT SUBSTR(name,1,3) FROM student;
TRIM() - SELECT TRIM(name) FROM student;
LENGTH() - SELECT LENGTH(name) FROM student;


25. NUMERIC FUNCTIONS
MOD() - SELECT MOD(15,4);
POWER() - SELECT POWER(2,3);
ROUND() - SELECT ROUND(345.345);
SIGN() - SELECT SIGN(-345.896);
SQRT() - SELECT SQRT(81);
TRUNCATE() - SELECT TRUNCATE(11.548,2);


26. AGGREGATE FUNCTIONS
MAX(), MIN(), COUNT(), SUM(), AVG()


27. DATE & TIME FUNCTIONS
CURDATE(), DATE(), MONTH(), YEAR(), NOW(), SYSDATE()


28. CONSTRAINTS
NOT NULL - prevents NULL values.
UNIQUE - ensures all values are unique.
PRIMARY KEY - unique and not null.
FOREIGN KEY - creates relationship between tables.
CHECK - ensures condition is met.
DEFAULT - sets default value.


29. JOINS
INNER JOIN - returns matching records.
LEFT JOIN - returns all from left + matched right.
RIGHT JOIN - returns all from right + matched left.
FULL JOIN - union of left and right joins.
SELF JOIN - joins table with itself.
NATURAL JOIN - joins with same column names.
CARTESIAN PRODUCT - all possible combinations.


30. AUTO_INCREMENT
Automatically increases column value for new records.

41. INDEX
CREATE INDEX myidx1 ON student (rollno);
ALTER TABLE student DROP INDEX myidx1;


42. GROUP BY
SELECT class, COUNT(*) FROM student GROUP BY class;


43. TRIGGERS
CREATE TRIGGER after_student_insert
AFTER INSERT ON student
FOR EACH ROW
INSERT INTO student_log VALUES (NEW.rollno, 'Record Added');


44. CASE STATEMENT
SELECT name, qty, total,
CASE
  WHEN total >= 20000 THEN 'Expensive Product'
  WHEN total >= 10000 THEN 'Medium Product'
  WHEN total >= 5000 THEN 'Average Product'
  ELSE 'Low Product'
END AS ProductStatus
FROM product;


45. UNION & INTERSECT
UNION - combines results, removes duplicates.
UNION ALL - includes duplicates.
INTERSECT - not directly supported in MySQL (use INNER JOIN).


46. TRANSACTIONS
SET autocommit = 0;
DELETE FROM student WHERE rollno > 101;
ROLLBACK;
COMMIT;


47. WINDOW FUNCTIONS
RANK(), DENSE_RANK(), ROW_NUMBER()
Example:
SELECT name, department, salary,
RANK() OVER (PARTITION BY department ORDER BY salary DESC) AS empRank
FROM employee;


48. EXPORT & IMPORT DATABASE
Export:
mysqldump -u root -p sql4pm > D:\SqlBackup\sql4pmBackup.sql

Import:
mysql -u root -p sql4pm < D:\SqlBackup\sql4pmBackup.sql
